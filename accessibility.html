Accessibility

Accessibility Statement

Last updated: May 2026

Our CommitmentTextured Talent serves professionals 50 and older. Accessibility is not a compliance checkbox — it is a core design requirement for the audience we serve.

Our Standards

We aim for WCAG 2.1 Level AA compliance across all pages on this website. This includes sufficient color contrast, keyboard navigation, screen reader compatibility, readable text sizes (nothing below 0.875rem / 14px), visible focus states on all interactive elements, and skip navigation links.

What We Have Done

  • All meaningful images include descriptive alt text
  • No text on this site is smaller than 0.875rem (14px) — the minimum recommended for a 50+ audience
  • All interactive elements have visible focus states
  • A skip link is available at the top of every page
  • Color is never used as the sole means of conveying information
  • All statistics are displayed inline and accessible — not hidden in hover states
  • Navigation is keyboard-accessible throughout, including mobile hamburger menus
  • ARIA labels are used on navigation, dialogs, dynamic content, and embedded tools
  • Section headings follow a logical hierarchy on all pages
  • Eyebrow labels on light backgrounds use deep mauve (#4A1F3E) at 12.6:1 contrast ratio
  • Body text meets a minimum 7.8:1 contrast ratio throughout
  • The Truth Mirror™ tool includes "You can stop at any time" messaging and privacy reassurance before data collection
  • OG and meta tags are present on every page for assistive technology and social sharing

Known Limitations

Scheduling widget: The Cal.com booking widget on the Consultation page is a third-party tool. While we have added descriptive ARIA labels to the iframe, the widget's internal keyboard navigation is governed by Cal.com's own implementation. If you experience difficulty, please email hello@texturedtalent.com directly to schedule your Showroom consultation.

Founder video: A founder video module appears as a placeholder on the homepage. When video is published, it will include closed captions, a written transcript linked near the player, and no-autoplay behavior. The placeholder is marked with appropriate ARIA attributes so screen readers understand it is not yet active content.

Orange text on light backgrounds: Burnt orange (#DF6E1B) achieves 3.1:1 contrast on cream backgrounds, which is below the 4.5:1 threshold for normal text. We address this by using orange exclusively at bold, 14px or larger sizes — qualifying as WCAG large text at the 3:1 threshold — and by using deep mauve (#4A1F3E) for all small-size labels. We are monitoring this and will darken the orange value if future testing identifies issues.

The Truth Mirror™ tool: This page uses a standalone interface with its own color system. All text has been verified at or above 0.875rem. The quiz includes stop-at-any-time messaging, a privacy note before data collection, and keyboard-navigable answer options.

Ongoing Improvement

We conduct accessibility reviews with each major site update. Our planned next steps include a formal Lighthouse and axe accessibility audit, a full keyboard navigation test across all pages, and a screen-reader walkthrough of The Truth Mirror™ tool. Results will be used to update this statement and address any gaps identified.

Feedback

If you encounter any accessibility barriers on this site, please contact us at hello@texturedtalent.com. We take these reports seriously and will address them promptly.
